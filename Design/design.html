Google Maps

Advanced Markers (Pin)

"Use markers to display single locations on a map. This guide shows you how to use advanced markers.
With advanced markers you can create and customize highly performant markers,
and make accessible markers that respond to DOM click events and keyboard input."

A este marker default el programa puede:

  • Scale the marker.
  • Change the background color.
  • Change the border color.
  • Change the glyph color.
  • FontAwesome Icon for the glyph.
  • Hide the glyph.

Graphic Markers

  • ¡Fondo Transaprente!
  • ¡PNG format is recommended!
  • Chico: 20px ancho / 32px alto, Grande: 56px X 56px, muy grande 64px X 64px.
    Bonito, indicativo, sirva, clickable (en su caso), pero ¡no tape cosas del mapa!
  • Recomiendan 20-40 pixels in width / height so you should aim to create your images with dimensions of 60-80 pixels or around 120px and scale them.
